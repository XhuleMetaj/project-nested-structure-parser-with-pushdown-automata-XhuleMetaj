Non-CFL Limitation

Our PDA handles Dyck-style mixed bracket matching — this is exactly context-free. However, if we required the total count of ( to equal the total count of [ globally (e.g. the language anbncn), this would be beyond CFL and would require a Turing machine. Our design intentionally stays within this well-defined CFL boundary.

Correctness Proof (Sketch)

Claim

PDA M accepts exactly the language L of strings whose mixed delimiters (), [], {} are properly nested, whose non-delimiter symbols belong to the restricted math alphabet Σ, and whose escape sequences \x treat x as a literal.

Soundness — every accepted string ∈ L

Invariant I: At any step in q₀, the stack (above $) is a sequence of opening delimiters appearing in the input prefix in order, each corresponding to an unmatched opener.
1
Base case

Initially stack = {$}; no openers yet — invariant holds trivially.

2
Push step (open delimiter)

Reading ( [ { appends the matching opener to the stack — invariant maintained.

3
Pop step (close delimiter)

Reading a closer removes the top only if it is the matching opener; otherwise → q_reject. We never pop a non-matching symbol, so invariant is preserved on success.

4
Escape step

In q_esc, one character is consumed with no stack change. Delimiters in escaped form cannot affect Invariant I.

5
Literal step

Math characters (digit, letter, op, ws) do not alter the stack — invariant trivially maintained.

6
Conclusion

On acceptance, input is empty and stack = {$}, so every opener was matched. Illegal symbols cause rejection, so accepted strings use only allowed terminals. Therefore every accepted string satisfies L.

Completeness — every w ∈ L is accepted

1
Well-nested delimiters

Each opener in w triggers a push. Since w is well-nested, when a closer appears the stack top is exactly its partner, so the PDA pops instead of rejecting.

2
Escape sequences

Each pair \x moves to q_esc and consumes x without stack change — legal in L.

3
Math terminals

Characters only advance the read head, leaving the stack intact.

4
Termination

At end of input, no unmatched openers remain (w ∈ L), so stack = {$} and the PDA enters q_accept. Therefore w is accepted. ∎
